# My Calculator

A simple calculator built using C++.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Square
* Square Root

## Concepts Used

* C++
* User Input/Output
* If-Else Statements
* Mathematical Functions (`pow()`, `sqrt()`)

## How to Run

1. Compile the code:

   ```
   g++ calculator_code.cpp -o calculator_code
   ```
2. Run the program:

   ```
   ./calculator_code
   ```
